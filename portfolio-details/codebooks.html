Description:

Code and markup editor web app.  Code editor bundles html, css, javascript, and files imported from unpkg.com. Bundled code is displayed in a preview window or an error message alerting to any issues.

Deployed on an AWS ec2 instance using nginx, docker, and Route 53 for dns.  Styled using bulmaswatch superhero.
